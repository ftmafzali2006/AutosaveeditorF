# AutoSave Editor

این پروژه یک ویرایشگر متن ساده است که امکانات زیر را دارد:

- ایجاد فایل جدید
- باز کردن فایل موجود
- ذخیره فایل و ذخیره با نام جدید
- ذخیره خودکار هر 30 ثانیه
- استفاده از کلیدهای ترکیبی برای ذخیره و باز کردن و ایجاد فایل جدید
- امکانات ویرایشی مانند Cut، Copy، Paste، Select All و Clear

این پروژه با استفاده از Windows Forms در زبان C# ساخته شده است.
